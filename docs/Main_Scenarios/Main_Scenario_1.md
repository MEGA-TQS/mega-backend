# Main Scenario 1  
## Search and Rent Equipment  

**Description**:  
A user needs sports equipment for temporary use and searches the platform to find a suitable item based on sport, location, and availability. Upon finding a match, they complete the rental process efficiently.

**Scenario Flow**:  
1. The user accesses the **homepage** via mobile app or web.  
2. The user enters a search query and applies filters: **sport**, **location**, **dates**, and optional preferences (e.g., price range, condition).  
3. The system displays a list of available items with key details: photos, daily rate, owner rating, and real-time availability.  
4. The user selects an item to view the **detailed listing**, including description, specifications, reviews, damage policy, and pickup/delivery options.  
5. The user submits a **rental request** for the selected dates, optionally adding insurance or accessories.  
6. The owner receives a **notification** with renter details and either accepts or declines the request.  
7. Upon acceptance, the user completes **secure payment** and receives confirmation with pickup/delivery instructions and a digital receipt.
