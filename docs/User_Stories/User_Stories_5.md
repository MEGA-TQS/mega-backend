# User Stories Related to Main Scenario 5 (Rent Multiple Items)

---

## US10 - Rent Multiple Items in One Booking:

- As a renter
- I want to rent multiple items in a single transaction
- So that I can organize group activities more easily.

#### AC:

- User can add several items to a single booking request.
- Availability is checked for all items.
- User pays once for all items.
