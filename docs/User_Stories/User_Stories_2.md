# User Stories Related to Main Scenario 2 (Equipment Management)

---

## US6 - Create Equipment Listing:

- As an owner
- I want to add a new equipment listing with photos and details
- So that renters can find and book my items.

#### AC:

- Owner can upload photos.
- Owner can enter description, condition, and category.
- Listing is saved and published.


## US7 - Set Price and Availability:

- As an owner
- I want to define the price per day and available dates
- So that renters know when and how much it costs to rent my gear.

#### AC:

- Owner can set a daily price.
- Owner can block unavailable dates.
- Availability updates appear in search results.
