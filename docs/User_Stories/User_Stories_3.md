# User Stories Related to Main Scenario 3 (Booking & Availability System) 

---

## US8 - Instant Booking:

- As a renter
- I want to instantly book selected items marked as Instant Book
- So that I can secure equipment without waiting for owner approval.

#### AC:

- Items eligible for instant booking are clearly labeled.
- User books directly and calendar is auto-blocked.
- Owner and renter receive immediate confirmation.
