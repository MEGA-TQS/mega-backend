# User Stories Related to Main Scenario 1 (Search and Rent Equipment)

---

## US1 - Search for Equipment:

- As a renter
- I want to search equipment by sport, location, and dates
- So that I can find available items that fit my needs.

#### AC:

- User can enter keywords.
- User can filter by sport, location, dates, price.
- Results only show available items.


## US2 - View Equipment Details:

- As a renter
- I want to view full details of an item
- So that I can understand its condition, specs, policies, and reviews.

#### AC:

- User sees photos, description, specs.
- User sees reviews and ratings.
- User sees pickup/delivery options.


## US3 - Request a Rental:

- As a renter
- I want to submit a rental request for specific dates
- So that the owner can approve or reject it.

#### AC:

- User selects dates.
- System validates availability.
- Request is sent to owner.


## US4 - Owner Accepts/Declines Rental:

- As an owner
- I want to quickly accept or decline rental requests
- So that I can control who rents my equipment.

#### AC:

- Owner receives notification.
- Owner sees renter details.
- Owner can accept/decline with one tap.


## US5 - Complete Rental Payment:

- As a renter
- I want to pay securely after approval
- so that I can confirm the booking.

#### AC:

- Payment succeeds or fails clearly.
- User receives confirmation + receipt.
